# Design

## Goal

The goal of this exercise is to teach the student the basics of programming in Odin.

## Learning objectives

- Learn about packages
  - How to define a package
  - How to import a package
- Learn about variables
  - How to define a variable
  - How to use type inference
- Learn about constants
  - How to define a constant
  - How to use type inference
- Know how to define a procedure
  - Know how to return a value from a procedure
  - Know how to call a procedure
- Know how to perform basic mathematical operations
- Know how to define comments

## Out of scope

- Naming rules for identifiers
- Default parameters for procedures
- Error handling
- Memory allocation
- Named return values

## Concepts

The Concept this exercise illustrate is `Basics`:

The `Basics` concept introduce the student to an Odin minimum program.
This covers comments, packages, variables, constants, and procedures.

## Prerequisites

There are no prerequisites.
